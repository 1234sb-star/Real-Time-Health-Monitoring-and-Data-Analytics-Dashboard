# 🏥 Real-Time Health Monitoring and Data Analytics Dashboard

## 📌 Project Overview
This project is a real-time health monitoring and data analytics dashboard built using Python (Flask) and Chart.js. It simulates live health data such as heart rate, steps, and calories, and visualizes it through interactive dashboards.

---

## 🚀 Features

- 📊 Real-time data visualization
- ❤️ Heart rate monitoring (Line Chart)
- 🚶 Steps tracking (Bar Chart)
- 🔥 Calories tracking (Doughnut Chart)
- 📈 Detailed analytics dashboard
- 📄 Health report (Average, Min, Max)
- 🌐 Multi-page navigation (Dashboard, Analytics, Report)
- 🎨 Responsive and modern UI

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML, CSS, JavaScript
- Chart.js
